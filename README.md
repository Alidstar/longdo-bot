## Welcome to Preo Bot Pages

Preo Bot is a preorder helper bot for Line platform.

### Development

Python

### Commands

```
1. !preorder <name>
2. !add <user> <order> <total>
3. !del <user> <order> <total>
4. !endorder
5. !list <name|ID>
6. !help 
```
